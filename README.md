# titanic-ml
Projet de classification binaire sur le dataest Titanic. Implémentation d'un modèle Randon Forest avec Scikit-Learn, incluant le prétraitement  des données, la gestion des hyperparamètres et une analyse critique des erreurs de prédiction.

# Analyse d'erreur
Le modèle atteint 80% d'accuracy. L'analyse du rappel (0.68 pour les survivants) montre que le modèle a tendance à classer à tort certains passagers comme décédés. Une étude plus poussée sur les variables 'Cabin' et 'Fare' permettrait d'affiner la précision sur les passagers de 1ère classe.
